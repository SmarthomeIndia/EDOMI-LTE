Administrationsseite***0-1Konfiguration***1000Visualisierung***1000-20Formatierung (Meldungsarchive)

In einer Visualisierung***b-0 können Inhalte von Meldungsarchiven***1000-60 mit Hilfe des Visuelements Meldungsarchiv***1002-25 angezeigt werden. Die Meldungen können bei Bedarf individuell formatiert werden (Farben und Symbol), indem einer Meldung eine hier konfigurierte Formatierung zugewiesen wird. Beim Hinzufügen einer Meldung in ein Meldungsarchiv (Befehle***1007) kann die gewünschte Formatierung ausgewählt werden. Alternativ kann im Inhalt der Meldung auf eine Formatierung verwiesen werden:
  • "<Formatierungs-ID>***<Meldung>"
  • Formatierungs-ID: die ID der gewünschten Formatierung
  • Meldung: der eigentliche Inhalt der Meldung
  • Beispiel: "34***EDOMI" erzeugt eine Meldung mit der Formatierungs-ID 34 und dem Inhalt "EDOMI"
  • Hinweis: Eine derart referenzierte Formatierung hat stets Priorität gegenüber einer per Befehl zugewiesenen Formatierung. Exisitiert eine referenzierte Formatierung nicht (mehr), wird die Meldung unformatiert ausgegeben.

Eigenschaften

  • Name: frei wählbarer Name der Formatierung
  • Symbol: Auswahl eines Bildes***1000-28, das als Symbol (Icon) in der Meldung angezeigt werden soll
  • Farbe: Auswahl einer Vordergrundfarbe***1000-26 für die Schriftfarbe der Meldung
    • Wichtig: Dynamische Farben werden nicht unterstützt!
  • Hintergrundfarbe: Auswahl einer Hintergrundfarbe***1000-25 als Hintergrund der Meldung
    • Wichtig: Dynamische Farben werden nicht unterstützt!
