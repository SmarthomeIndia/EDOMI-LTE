Administrationsseite***0-1Konfiguration***1000Kameras***1000-80Kameraarchive

Kameraarchive können eine beliebige Anzahl von Kamerabildern archivieren. Archivierte Kamerabilder werden z.B. in Kameraaufnahmen***104 oder mit Hilfe des Visuelements "Kameraarchiv"***1002-20 angezeigt.

Eigenschaften

  • Name: frei wählbarer Name des Kameraarchivs
  • Kamera: Auswahl der konfigurierten Kamera***1000-81, deren Bild archiviert werden soll
  • KO: Status
    • bei jeder Änderung des Datenbestandes an diesem Archiv wird dieses KO auf die Gesamtanzahl der Archiveinträge gesetzt:
    • beim Start von EDOMI
    • beim Hinzufügen/Entfernen eines Eintrags
    • beim automatischen Löschen von Einträgen (s.u.)
  • KO: letzte Archivierung
    • beim Hinzufügen eines Kamerabildes wird dieses KO mit Metadaten bezüglich des archivierten Bildes belegt
    • Diese Metadaten haben folgende Struktur und können z.B. von speziellen Logikbausteinen***r-0 ausgewertet werden:
      • <Archiv-ID>;<Kamera-ID>;<Datum>;<Uhrzeit>;<Mikrosekunden>;<Dateiname des Bildes mit Pfad>
  • Totzeit: legt fest, ob das Hinzufügen eines Eintrags jederzeit oder nur nach Verstreichen einer Mindestzeit seit dem letzten Hinzufügen ermöglicht wird
    • 0: das Hinzufügen eines Eintrags ist ohne Einschränkung möglich
    • 1..∞ (Sekunden, Ganzzahl): das Hinzufügen eines Eintrags ist nur alle 1..∞ Sekunden möglich
  • Speicherdauer: legt fest, ob und wann Einträge automatisch gelöscht werden
    • Archiveinträge werden grundstäzlich mit einem Zeitstempel archiviert
    • ist ein Archiveintrag älter als die festgelegte Speicherdauer, wird der Archiveintrag beim Start von EDOMI und täglich um 00:00:00 Uhr aus dem Archiv gelöscht

Hinzufügen und Entfernen von Archiveinträgen

Archiveinträge werden über entsprechende Befehle***1007 hinzugefügt oder entfernt. Zudem können Logikbausteine***r-0 weitere Funktionen zur Verwaltung von Archiveinträgen bereitstellen. Hinweis: Archiveinträge werden u.U. automatisch gelöscht, sofern die Eigenschaft "Speicherdauer" (s.o.) entsprechend konfiguriert wurde.

Struktur von Archiveinträgen

Archiveinträge werden jeweils mit einem eindeutigen Zeitstempel versehen (Datum, Uhrzeit und Mikrosekunden). Das eigentliche Kamerabild wird als JPG-Datei im Verzeichnis www/data/liveproject/cam/live abgelegt.
