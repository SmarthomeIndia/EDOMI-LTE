Basis-Konfiguration

Die grundlegende Konfiguration von EDOMI erfolgt über eine Textdatei. Die Datei selbst enthält bereits alle Informationen und Hilfestellungen zu den jeweiligen Parametern:
  • Dateiname: edomi.ini
  • Verzeichnis:
Die Konfigurationsdatei kann in der Verwaltung***103-6 oder mit jedem UTF-8-fähigen Texteditor bearbeitet werden. Wichtig: Änderungen an dieser Konfigurationsdatei werden erst nach einem Neustart des Servers übernommen (ein Neustart von EDOMI genügt nicht).

Wichtige Grundeinstellungen nach der Installation

Die folgenden Parameter müssen ggf. unmittelbar nach der Installation angepasst werden: Zeitzone:
  • set_timezone='Europe/Berlin'
  • Wichtig: Die hier angegebene Zeitzone sollte mit der Zeitzone des Betriebssystems übereinstimmen!
  • Beispiel: 'Europe/Berlin' = Deutschland
  • Weitere Zeitzonen-Einstellungen unter http://php.net/manual/en/timezones.php (externer Link)
IP-Adresse des EDOMI-Servers im lokalen Netzwerk:
  • Beispiele:
  • global_serverIP='192.168.0.10' - die IP-Adresse wird manuell auf 192.168.0.10 gesetzt
  • global_serverIP='' - die IP-Adresse wird automatisch ermittelt (bei jedem Server-Neustart)
  • Hinweis: Die IP-Adresse wurde i.d.R. bereits auf die entsprechende Eingabe während der Installation von EDOMI gesetzt.
Alle weiteren Parameter sind standardmäßig derart konfiguriert, dass EDOMI ohne Gefahr in Betrieb genommen werden kann (z.B. ist die Kommunikation mit dem KNX-Bus***a-1-1 standardmäßig deaktiviert).

Module

Im Abschnitt "Module" sind die folgenden Funktionseinheiten konfigurierbar, bzw. lassen sich aktivieren oder deaktivieren:
  • Email-Gateway***a-1-0: Senden von Emails, z.B. Systemmitteilungen oder individuelle Email-Benachrichtigungen
  • KNX-Gateway***a-1-1: Erforderlich für die Kommunikation mit dem KNX-Bus
  • Telefonie-Gateway***a-1-2: ermöglicht die Nutzung von Telefonie-Funktionen mit Hilfe einer Fritzbox (AVM)
  • Anrufmonitor***a-1-3: Überwachung von ein- und ausgehenden Anrufen (benötigt das Telefonie-Gateway)
  • IRtrans***a-1-4: ermöglicht das Einlernen und Senden von IR-Befehlen (Infrarot-Fernbedienungen) mit Hilfe eines IRtrans-Gerätes
  • Digitaler Videorekorder***a-1-5: ermöglicht die fortlaufende Aufzeichnung von Einzelbildern entspechend konfigurierter Kameras
