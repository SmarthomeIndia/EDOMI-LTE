String: KO-Werte einsetzen
▸ E1TriggerA1 ▸
Dieser Baustein ersetzt beliebig viele Platzhalter (Variablen) durch die entsprechenden KO-Werte. Die nachfolgenden Variablen werden bei jedem Triggern des Bausteins durch einen String ≠[leer] an E1 durch den aktuellen KO-Wert ersetzt. Das Ergebnis wird an A1 ausgegeben:
  • "{id}": Repräsentiert den Wert eines KOs mit dieser ID
    • z.B. "Die aktuelle Server-IP ist {3}" wird zu "Die aktuelle Server-IP ist 1.2.3.4"
    • Hinweis: Existiert das KO nicht, wird der Ausdruck unverändert ausgegeben.
  • "{x/y/z}": Repräsentiert den Wert einer KNX-GA
    • z.B. "Temperatur = {1/2/3} Grad Celsius" wird zu "Temperatur = 23.5 Grad Celsius"
    • Hinweis: Existiert die KNX-GA nicht, wird der Ausdruck unverändert ausgegeben.
Hinweis: In einem String kann eine beliebige Anzahl der o.g. Variablen angegeben werden. E1: String (≠[leer] Triggert den Baustein) A1: Ergebnis als String (maximal 10.000 Zeichen, überzählige Zeichen werden abgeschnitten)
