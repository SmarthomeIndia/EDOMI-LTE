Dieser Baustein gibt an den Ausgängen bei jedem Triggern (E1) die unten genannten Werte aus. Getriggert wird der Baustein über ein neues Telegramm ≠[leer] an E1. Anwendungsbeispiel: An E1 kann z.B. das System-KO[5] (Systemzeit) angelegt werden, um sekündlich aktuelle Werte an den Ausgängen zu erhalten. Alternativ kann z.B. das System-KO[26] (Trigger: Minütlich) an E1 angelegt werden, zusätzlich kann ein Initialwert 1 angegeben werden - dies führt dazu, dass der Baustein beim Start (Initialwert) und anschließend zu jeder vollen Minute getriggert wird. Weitere Informationen können ggf. der Hilfe zur Statusseite***0-0 entnommen werden. Hinweis: Wenn das Modul Anrufmonitor***a-1-3 nicht aktiviert ist, erfolgt keine Reaktion an den Ausgängen. E1: jedes Telegramm ≠[leer] triggert den Baustein A1: Verbindungsstatus zur Fritzbox (Anrufmonitor): 1=verbunden, 0=nicht verbunden A2: Anzahl der eingegangenen Anrufe A3: Anzahl der ausgegangenen Anrufe A4: Speichernutzung des Prozesses in MB
