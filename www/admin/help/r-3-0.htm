Entwicklung von Visuelementen

Visuelemente***1002 (VSE) werden durch eine Datei repräsentiert, die aus Deklarationen sowie PHP- und Javascript-Funktionen besteht. Grundsätzlich stellt eine solche Datei folgende Informationen bzw. Scripte bereit:
  • Deklarationen (z.B. Name, diverse Defaultparameter, etc.)
  • Eigenschaften-Dialog zur Bearbeitung der Parameter im Visueditor
  • ein optionales Script, das bei der Projekt- oder Visuaktivierung angewendet wird
  • Scripte für die Darstellung des Visuelements im Visueditor
  • Scripte für die Darstellung/Funktionalität des Visuelements in der Visualisierung
  • ein optionales Script für die Verwendung im Visueditor und(!) in der Visualisierung (ähnlich einer Inklude-Datei)
  • Hilfe
