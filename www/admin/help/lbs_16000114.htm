▸ E2Dauer: lang (ms)500langA2 ▸
Dieser Baustein unterscheidet zwischen einem langen und einem kurzen Signal ≠0 an E1. Wird E1 auf einen Wert ≠0 gesetzt, beginnt die Zeitmessung. Wird anschließend E1=0 gesetzt, wird entweder A1=1 gesetzt (kurz) oder A2=1 gesetzt (lang). Wird die Dauer an E2 erreicht, ohne(!) dass E1=0 gesetzt wird, wird dennoch A2=1 gesetzt (lang). E1: 0=Aus, ≠0=Ein E2: Dauer in Millisekunden für eine lange Signaldauer A1: 1=Kurzes Signal A2: 1=Langes Signal
