Kameraansichten ermöglichen die Anzeige des Kamerabildes einer konfigurierten Kamera***1000-81 in einer Visualisierung. Weiterhin können Kameraansichten auch beim Einsehen von Kameraarchiven bzw. Aufnahmen des Digitalen Videorekorders (Kameraaufnahmen***104) verwendet werden. Eine Kameransicht ermöglicht optional das Festlegen eines Bildausschnitts bzw. das Entzerren des Bildes einer 360-Grad-Kamera ("Fisheye-Dewarping"). Hinweis: Für die Anzeige eines Live-Kamerabildes in einer Visualisierung ist immer eine konfigurierte Kameraansicht erforderlich, auch wenn keine Bildverarbeitung (Ausschnitt bzw. Fisheye-Dewarping) erfolgen soll.

Eigenschaften

  • Name: frei wählbarer Name der Kameransicht
  • Kamera: Auswahl der konfigurierten Kamera***1000-81, für die eine Kameraansicht generiert werden soll
  • Bildbearbeitung: ermöglicht optional das Festlegen eines Bildausschnitts bzw. das Entzerren des Bildes einer 360-Grad-Kamera ("Fisheye-Dewarping")
    • deaktiviert (Originalbild): es erfolgt keine Bearbeitung, d.h. das Kamerabild wird unverändert ausgegeben
    • Ausschnitt: es kann ein beliebiger Bildausschnitt zur Ausgabe festgelegt werden
    • 360-Grad-Entzerrung (Kreis): ein Bildausschnitt einer 360-Grad-Kamera wird entzerrt ausgegeben, das Originalbild der Kamera ist kreisrund (unabhängig von dem Seitenverhältnis des Kamerabildes)
    • 360-Grad-Entzerrung (Ellipse): ein Bildausschnitt einer 360-Grad-Kamera wird entzerrt ausgegeben, das Originalbild der Kamera ist ellipsenförmig verzerrt (unabhängig von dem Seitenverhältnis des Kamerabildes)
  • Einstellungen: falls eine Bildbearbeitungsoption gewählt wurde, öffnet diese Schaltfläche das Fenster "Kameraansicht: Einstellungen***1020" zur Festlegung weiterer Parameter
Wichtig: Das Festlegen eines Bildausschnitts bzw. das Entzerren eines 360-Grad-Kamerabildes ist sehr rechenintensiv (die Berechnungen werden auf dem Endgerät/Client ausgeführt, nicht auf dem EDOMI-Server). Nach Möglichkeit ist also auf eine Bildbearbeitung zu verzichten, es erfolgen dann keine weiteren Berechnungen und das Kamerbild wird im Originalzustand ausgegeben.
