Administrationsseite***0-1Konfiguration***1000Emails***1000-120Eigene Emails

Eigene Emails sind vorkonfigurierte Emails***1000-120, die mit entsprechenden Befehlen***1007 nach Belieben versendet werden können.

Eigenschaften

  • Name: frei wählbarer Name der Email-Vorlage
  • Empfänger: Email-Adresse des Empfängers
    • wird keine Empfänger-Adresse angegeben, wird die Standard-Adresse aus der Basis-Konfiguration***a-1 übernommen
    • Hinweis: Die maximale Gesamtlänge beträgt 1000 Zeichen.
  • Betreff: Betreffzeile der Email
    • optional können auch spezielle Variablen angegeben werden (s.u.)
    • Hinweis: Die maximale Gesamtlänge beträgt 1000 Zeichen.
  • Inhalt: Inhalt der Email
    • optional können auch spezielle Variablen angegeben werden (s.u.)
    • Hinweis: Die maximale Gesamtlänge beträgt 10.000 Zeichen.

Verwendung von Variablen

In den Feldern "Betreff" und "Inhalt" können nachfolgende Variablen verwendet werden (ohne "" anzugeben):
  • "{DATETIME} ": aktueller Zeitstempel
    • z.B. "Es ist {DATETIME} Uhr." wird zu "Es ist 27.03.2014 12:01:34 Uhr."
  • "{id}": Repräsentiert den Wert eines KOs mit dieser ID
    • z.B. "Die aktuelle Server-IP ist {3}" wird zu "Die aktuelle Server-IP ist 1.2.3.4"
    • Hinweis: Existiert das KO nicht, wird der Ausdruck unverändert ausgegeben.
  • "{x/y/z}": Repräsentiert den Wert einer KNX-GA
    • z.B. "Temperatur = {1/2/3} Grad Celsius" wird zu "Temperatur = 23.5 Grad Celsius"
    • Hinweis: Existiert die KNX-GA nicht, wird der Ausdruck unverändert ausgegeben.

Senden von Emails

Eigene Emails werden über den entsprechenden Befehl***1007 versendet.
