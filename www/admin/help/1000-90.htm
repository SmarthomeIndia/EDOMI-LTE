Administrationsseite***0-1Konfiguration***1000Sequenzen

Mit Hilfe von Sequenzen können beliebige Befehle***1007 in einer definierten und anpassbaren Reihenfolge ausgeführt werden. Jede Sequenz kann eine beliebige Anzahl an Befehlen verwalten. Nach der Ausführung eines Befehls kann optional eine individuelle Wartezeit bis zur Ausführung des nächsten Befehls konfiguriert werden. Achtung: Sequenzen können wiederum Sequenzen ausführen, dabei ist jedoch mit besonderer Sorgfalt vorzugehen um Endlosschleifen zu vermeiden!

Eigenschaften

  • Name: frei wählbarer Name der Sequenz

Schaltflächen

Mit der Schaltfläche "Sequenz erstellen" wird eine neue Sequenz mit den o.g. Eigenschaften angelegt. Anschließend stehen weitere Eigenschaften zu Verfügung:

Weitere Eigenschaften

  • Befehle: hier werden die Befehle***1007 dieser Sequenz verwaltet
    • Für jeden Befehl kann eine Wartezeit (in Sekunden) bis zur Ausführung des nächsten Befehls angegeben werden:
      • 0: nicht warten, d.h. der nachfolgende Befehl wird unmittelbar ausgeführt
      • 1..∞: nach Ausführung dieses Befehls wird x Sekunden gewartet, bis der nachfolgende Befehl ausgeführt wird
      • Hinweis: Befehle werden asynchron ausgeführt, d.h. die Wartezeit schließt die tatsächliche Ausführungsdauer eines Befehls nicht(!) mit ein.
    • Hinweis: Per Kontextmenü (rechte Maustaste) kann u.a. die Reihenfolge der Befehle angepasst werden.
