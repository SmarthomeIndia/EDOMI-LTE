Administrationsseite***0-1Konfiguration***1000Zeitschaltuhren

Zeitschaltuhren setzen ein Kommunikationsobjekt zu einem definierten Zeitpunkt (Schaltzeit) auf einen Wert. Jede Zeitschaltuhr kann eine beliebige Anzahl an Schaltzeiten***1019 verwalten.

Eigenschaften

  • Name: frei wählbarer Name der Zeitschaltuhr
  • KO: Steuerung der Zeitschaltuhr
    • mit diesem KO wird die Zeitschaltuhr ein- oder ausgeschaltet
    • ein KO-Wert="0" (bzw. ≠"1") deaktiviert die Zeitschaltuhr, ein KO-Wert="1" aktiviert die Zeitschaltuhr
    • Hinweis: Dieses KO wird ggf. vom Visuelement Zeitschaltuhr***1002-22 zum Aktivieren/Deaktivieren der Zeitschaltuhr gesetzt bzw. ausgewertet.
  • KO: Zusatzbedingung
    • dieses (optionale) KO kann als weitere Filterbedingung einer Schaltzeit verwendet werden
    • für jede Schaltzeit***1019 kann festgelegt werden, ob dieses KO ignoriert wird, das KO den Wert "1" haben muss oder das KO nicht den Wert "1" haben darf
    • Hinweis: Die Zusatzbedingung (KO-Wert) wird intern UND-verknüpft, muss also ggf. erfüllt sein, damit die Schaltzeit wirksam wird.

Schaltflächen

Mit der Schaltfläche "Zeitschaltuhr erstellen" wird eine neue Zeitschaltuhr mit den o.g. Eigenschaften angelegt. Anschließend stehen weitere Eigenschaften zu Verfügung: Die Schaltfläche "Vorschau" öffnet ein neues Fenster mit einer Vorschau für alle konfigurierten Schaltzeiten (das KO "Zusatzbedingung" wird dabei ignoriert).

Weitere Eigenschaften

  • Schaltzeiten: hier werden die Schaltzeiten***1019 für diese Zeitschaltuhr verwaltet
    • Hinweis: Die hier konfigurierten Schaltzeiten sind in einer Visualisierung nicht veränderbar. Jedoch können ggf. in einer Visualisierung mit Hilfe des Visuelements Zeitschaltuhr***1002-22 weitere Schaltzeiten ergänzt werden.
  • Makro-Vorgaben für Visualisierungen: hier müssen sämtliche Makros***1000-95 für diese Zeitschaltuhr definiert werden, die in einer Visualisierung verwendet werden sollen
    • die hier aufgelisteten Makros werden beim Erstellen einer Schaltzeit in einer Visualisierung zur Auswahl angeboten
    • Hinweis: In einer Visualisierung werden ausschließlich die hier aufgelisteten Makros zur Auswahl angeboten. Die in der Administration konfigurierten (fixen) Schaltzeiten sind hiervon unabhängig und können sämtliche zur Verfügung stehenden Makros ausführen.

Funktionsweise der Zeitschaltuhr

Die Zeitschaltuhr arbeitet wie ein Terminkalender, d.h. ein zugewiesenes Makro wird ausgeführt, sobald eine Schaltzeit***1019 erreicht wird (und die weiteren Filterbedingungen erfüllt sind). Ist die Zeitschaltuhr deaktiviert, werden alle Schaltzeiten ignoriert. Beim Aktivieren der Zeitschaltuhr wird nicht(!) überprüft, ob eine Schaltzeit vor(!) dem Zeitpunkt der Aktivierung konfiguriert wurde: Das Makro wird erst beim Erreichen der nächsten(!) Schaltzeit ausgeführt. Die Zeitschaltuhr arbeitet intern sekundengenau, das kleinste Intervall zwischen zwei Schaltzeiten beträgt jedoch 1 Minute (Schaltzeiten werden in der Form hh:mm angegeben). Die Toleranzbreite beträgt maximal +1 Sekunde, wird z.B. eine Schaltzeit "15:00" Uhr konfiguriert, wird die Zeitschaltuhr frühstens um 15:00:00 Uhr und spätestens um 15:00:01 Uhr das Makro ausführen. Überschneidungen von Schaltzeiten werden nicht(!) überprüft oder eliminiert, bei der Konfiguration von Schaltzeiten ist ggf. darauf zu achten, dass sich die Schaltzeiten nicht gegenseitig beeinflussen oder zu widersprüchlichen Schalthandlungen führen.
