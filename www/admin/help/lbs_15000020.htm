Schwellenwert 15000020
Schwellenwert
▸ E1TriggerA1 ▸
▸ E2untere Schwelle 
▸ E3obere Schwelle 
Dieser Baustein vergleicht einen Ist-Wert (E1) mit zwei Soll-Werten (E2 und E3). Erreicht (bzw. über- oder unterschreitet) der Ist-Wert einen der Soll-Werte, wird A1 auf 1 bzw. 0 gesetzt. Wichtig: A1 wird bei jedem(!) Erreichen oder Über- bzw. Unterschreiten der Schwellenwerte gesetzt! Ggf. ist an A1 also ein "Send-By-Change"-Baustein einzusetzen. E1: Signal E2: untere Schwelle (ist E1 <= E2, wird A1=0 gesetzt) E3: obere Schwelle (ist E1 >= E3, wird A1=1 gesetzt) A1: 0=untere Schwelle erreicht, 1=obere Schwelle erreicht
